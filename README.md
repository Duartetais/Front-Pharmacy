Uma aplicação moderna de gestão farmacêutica desenvolvida com foco em **performance**, **escalabilidade** e **experiência do usuário**.

A **BioCore** foi criada para oferecer uma experiência intuitiva, moderna e eficiente no gerenciamento de categorias e departamentos da área da saúde, utilizando uma interface responsiva integrada com API REST.

# ⚙️ Como Executar o Projeto

## 1️⃣ Clone o repositório

```bash
git clone https://github.com/Duartetais/Front-Pharmacy.git
```

## 2️⃣ Acesse a pasta do projeto

```bash
cd Front-Pharmacy
```

## 3️⃣ Instale as dependências

```bash
npm install
```

## 4️⃣ Execute o projeto

```bash
npm run dev
```
